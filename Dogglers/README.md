# Dogglers

Independent project for Android Basics in Kotlin final pathway of Unit 2. This app shows scrolling
lists of dogs in three layout: horizontal, vertical, and grid.

## Topics covered in this module

- RecyclerView
- Layout manager(s)
- UI tests
